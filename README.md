# 🏨 P-ZED Homes – Hotel Management System

P-ZED Homes is a high-performance hotel management system built to bridge the gap between hospitality operations and financial precision.

It manages everything from guest booking to inventory and accounting, ensuring that every transaction is recorded accurately and every role operates efficiently.


## 🚀 Features

- Role-based dashboards (Owner, Manager, Receptionist, Housekeeper, Kitchen Staff, Bartender, Accountant, HR, Security, Cleaner)
- Guest booking and landing flows
- Supabase authentication and profile management
- Real-time operational dashboards (bookings, attendance, KPIs)
- Modules: Housekeeping, Kitchen/Bar, Inventory, Finance, HR, Maintenance, Reporting, POS, Communications
- Responsive UI for mobile, tablet, and desktop (web)


## 🔑 System Highlights

- ✅ Transaction-safe inventory flow (prevents duplicate or lost stock updates)  
- ✅ Role-based access control with clear separation of responsibilities  
- ✅ Responsive and consistent UI across all screen sizes  
- ✅ Optimized Flutter web performance for fast loading experience  
- ✅ Clean architecture for scalability and maintainability  


## 🛠 Tech Stack

- **Frontend:** Flutter 3.35 (Web + Mobile)
- **Backend:** Supabase (PostgreSQL, Auth, RLS, Edge Functions)
- **State Management:** Provider
- **Navigation:** `go_router`
- **Payments:** Bank transfer + Paystack integration
- **Design:** Custom responsive utilities + centralized theme system
